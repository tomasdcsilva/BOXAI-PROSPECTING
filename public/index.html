Como obter a chave Google Places API (gratuita)

A API do Google Places inclui 200 USD/mês em créditos gratuitos — suficiente para dezenas de pesquisas mensais.

  1. Acede a console.cloud.google.com
  2. Cria um projeto novo (ex: "BoxAI Prospeção")
  3. Vai a APIs & Services → Enable APIs e ativa: Places API e Geocoding API
  4. Vai a Credentials → Create Credentials → API Key
  5. Copia a chave e cola abaixo
